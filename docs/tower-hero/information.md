# Info commands

## q!`<tower>`

-   **usage: `q!<tower> <path>`**

-   **e.g. `q!ice 052`**

## q!`<hero>`

-   **usage: `q!<hero> <level>`**

-   **e.g. `q!quincy 20`**

## q!balance

-   **usage: `q!balance <tower/tower_path/tower_upgrade/hero> [version] [version]`**

-   Examples: 
   
    `q!balance heli`

    `q!balance heli#middle-path`

    `q!balance icicle_impale`

    `q!balance quincy`

    `q!balance sub#mid v15 v18`
