# Before you begin...

"alias" = "synonym"; "argument" = "input"

## `<stuff>`

`<stuff>` represents a **compulsory** input, you can see the input in the examples section

## `[stuff]`

`[stuff]` represents an **optional** input.

## aliases

q!alias can be used for both command aliases and input aliases (argument aliases).

-   **usage: `q!alias <term>`**
-   **examples: `q!alias wizard**

:::warning
Input alias may not match command alias.
For example, command aliases to invoke `q!wizard` might be different than argument aliases to invoke `q!2mp wizard`
