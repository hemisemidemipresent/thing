# Round commands

commands about round information

## q!round

displays round information

-   **usage: `q!round <round number>`**
-   **example: `q!round 36`**

## q!rbe

displays the number of pops on a certain round. Similar to [q!income](./income_calculator.mdx)

::: warning

BTD6 sometimes has bugs that overcount/undercount the number of pops, so dont expect 100% accuracy in-game

:::

-   **usage: `q!rbe <round1> <round2>`**
-   **examples: `q!rbe 36 48` or `q!rbe 78`**

## q!bloon

displays occurences of a certain type of bloon

-   **usage: `q!bloon <bloon>`**
-   **examples: `q!bloon cpurple`**
